# japan-trip-assets


这里是存储视频的仓库。

使用方法：

使用cdn进行展示。例如视频 1772a1837046ca4a8c0a7a0fc36d4812.mp4，可以使用：

```html
<video width="75%" controls preload="metadata">
  <source src="https://cdn.jsdelivr.net/gh/HaibinLai/japan-trip-assets@main/1772a1837046ca4a8c0a7a0fc36d4812.mp4" type="video/mp4">
</video>
```
